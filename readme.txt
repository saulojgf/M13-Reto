Desafío

Estás trabajando en el desarrollo de la página de inicio de un sitio web para un cliente que 
busca destacar su nueva línea de productos. La tarea principal es crear un componente de tarjeta 
que muestre de manera atractiva cada producto. Este componente debe ser completamente responsive 
y mostrar detalles adicionales de cada producto en un modal. El equipo ha optado por la metodología 
BEM para mantener un código organizado y escalable. Además, se sugiere el uso de Sass para estilizar 
el componente y la integración de Bootstrap para aprovechar la funcionalidad del modal. 
Considera que gestionarás el código utilizando Git y GitHub, así que no olvides tener instalado 
git y tener una cuenta en GitHub.
